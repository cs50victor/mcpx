===================================
Date: 2026-01-16
Context: Research for mcpx CLI upgrade to discover official MCP servers

PROBLEM STATEMENT
-----------------
When searching for MCP servers (e.g., "github", "slack"), existing registries return
third-party wrappers or hosted versions instead of official vendor implementations.

Example: `mcpx search github` returns Smithery's wrapper at `server.smithery.ai/github/mcp`
instead of GitHub's official server at `api.githubcopilot.com/mcp/`

KEY FINDINGS
------------
1. 97 official MCP servers exist on GitHub (search: "official mcp server")
2. No registry comprehensively indexes all official vendor servers
3. Smithery registry actively misleads - returns wrappers instead of officials
4. Docker's registry is best but incomplete (318 of 97+ officials)
5. Official servers use vendor domains (api.githubcopilot.com, googleapis.com)
6. Transport: Streamable HTTP for remote, stdio for local


RECOMMENDED SOLUTIONS
---------------------

Option 1: Use Docker's registry as base + GitHub API search
- Query docker/mcp-registry for existing configs
- Supplement with `gh search repos "official mcp server"` for discovery
- Pros: Structured YAML, actual repo links
- Cons: Need to merge two sources

Option 2: Build aggregator that queries multiple sources
- Docker registry (YAML configs)
- GitHub API (official server discovery)
- Optionally parse awesome-mcp-servers for üéñÔ∏è entries
- Pros: Comprehensive
- Cons: Complex maintenance

Option 3: Direct GitHub/npm search in CLI
- `mcpx search --github <query>` -> gh search repos "mcp server org:<query>"
- `mcpx search --npm <query>` -> npm registry search
- Pros: Always fresh data
- Cons: No curation, user must filter


GITHUB CLI COMMANDS FOR DISCOVERY
---------------------------------
# Find official MCP servers
gh search repos "official mcp server" --sort stars --limit 100 --json fullName,description,stargazersCount

# Search specific org
gh search repos "mcp-server org:github" --sort stars

# Search npm for MCP servers
curl -s "https://registry.npmjs.org/-/v1/search?text=mcp-server+scope:modelcontextprotocol"


DOCKER REGISTRY QUERY EXAMPLES
------------------------------
# List all servers
gh api repos/docker/mcp-registry/contents/servers --jq '.[].name'

# Get specific server config
gh api repos/docker/mcp-registry/contents/servers/notion/server.yaml --jq '.content' | base64 -d

# Server YAML format:
# name: notion
# image: mcp/notion
# source:
#   project: https://github.com/makenotion/notion-mcp-server
#   commit: <sha>
# config:
#   secrets: [...]
